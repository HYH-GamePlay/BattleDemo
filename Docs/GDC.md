## 游戏概述

### 基本信息
| 项目 | 内容 |
|------|------|
| **游戏名称** | （待定） |
| **游戏类型** | 俯视角即时战斗 + 肉鸽闯关 |

### 一句话描述
一款俯视角快节奏动作肉鸽，以创新的防御反击战斗和深度Build构建为核心，在美式卡通风格的线性关卡中体验黑暗奇幻剧情驱动的闯关冒险。

### 核心体验关键词
- **快节奏动作** - 强调走位、闪避、时机把握
- **防御反击** - 格挡、闪避、反击窗口的攻防博弈
- **Build构建** - 局内随机技能/遗物组合流派
- **永久成长** - 局外属性、武器、剧情能力解锁
- **剧情驱动** - 黑暗奇幻世界观，关卡推进叙事

## 核心玩法

### 核心循环
```
┌─────────────────────────────────────────────────────────┐
│                                                         │
│   战斗 ──→ 获取随机技能/遗物 ──→ 构建Build ──→ 推进关卡  │
│     │                                           │       │
│     │                                           ↓       │
│     │                                         死亡      │
│     │                                           │       │
│     └─────────── 永久成长系统 ←─────────────────┘       │
│                                                         │
└─────────────────────────────────────────────────────────┘
```

### 单局流程
1. **开始** - 选择已解锁的武器/初始能力
2. **关卡推进** - 通过线性关卡，遭遇精英怪+小怪组合
3. **Build构建** - 击败敌人获取随机技能/遗物
4. **Boss战** - 每关结尾Boss战
5. **死亡/通关** - 结算并保留永久成长进度

## 战斗系统

### 战斗设计理念
- **核心深度来源**：防御反击系统 + 基础攻击组合
- **战斗节奏**：快节奏，强调攻防博弈
- **操作手感**：流畅、响应迅速、反馈明确

### 基础攻击系统
| 操作 | 描述 |
|------|------|
| **轻击** | 快速攻击，低伤害，可连击 |
| **重击** | 慢速攻击，高伤害，可蓄力 |
| **蓄力攻击** | 长按重击，更高伤害+特殊效果 |

### 防御反击系统
| 操作 | 描述 | 设计要点 |
|------|------|----------|
| **防御** | 按住防御键格挡伤害 | 减伤50%，消耗耐力 |
| **格挡** | 受击瞬间防御 | 完美格挡，触发反击窗口 |
| **闪避** | 快速位移躲避攻击 | 无敌帧，消耗耐力 |
| **反击** | 格挡成功后限时输入 | 高伤害反击，反击窗口0.5秒 |

### 武器系统
| 武器类型 | 特性 |
|----------|------|
| **剑** | 平衡型，攻速适中 |
| **斧** | 高伤害，慢攻速 |
| **锤** | 最高伤害，击退效果 |
| **弓** | 远程攻击，需瞄准 |
| **枪** | 远程攻击，连射 |
| **法杖** | 魔法攻击，技能强化 |

### 敌人设计
| 敌人类型 | 设计要点 |
|----------|----------|
| **小怪** | 数量多、血量低、攻击模式简单 |
| **精英怪** | 数量少、血量中等、特殊机制 |
| **Boss** | 每关一个，多阶段、独特机制 |

## 肉鸽系统

### 局内随机要素

#### 技能/遗物系统
- **获取方式**：击败敌人、通关奖励、商店购买
- **稀有度**：普通 → 稀有 → 史诗 → 传说
- **类型分类**：
  - **攻击型**：伤害提升、攻击特效
  - **防御型**：减伤、护盾、回复
  - **辅助型**：移速、冷却、资源
  - **特殊型**：独特机制、Build核心

#### Build流派示例
| 流派 | 核心遗物 | 玩法特点 |
|------|----------|----------|
| **暴击流** | 暴击率+暴击伤害遗物 | 高风险高回报 |
| **反击流** | 格挡增强+反击伤害遗物 | 防守反击 |
| **攻速流** | 攻速+连击遗物 | 高频输出 |
| **技能流** | 技能冷却+技能伤害遗物 | 技能为主 |

### 局外永久成长

#### 成长内容
| 类型 | 内容 |
|------|------|
| **属性升级** | 攻击力、生命值、耐力、移速等 |
| **武器解锁** | 新武器类型、武器皮肤 |
| **剧情能力** | 剧情推进解锁的特殊能力 |

#### 成长节奏
- 每次死亡获得"成长点数"
- 成长点数用于解锁永久升级
- 升级幅度适中，不影响核心挑战
